# Arch-Linux

Instalacija Arch linuxa v slovenščini. Izbiraš lahko med:

KDE + kdm

PLASMA 5 + sddm

GNOME + gdm

MATE + lxdm

CINNAMON + gdm

Xfce + lxdm

LXDE + lxdm

ENLIGHTENMENT + slim


Navodilo:

1.) Boot Arch linux CD

2.) loadkeys slovene

3.) wget https://raw.githubusercontent.com/puikonen/Arch-Linux/master/AIS_sl.sh && sh AIS_sl.sh

4.) sledi navodilom
